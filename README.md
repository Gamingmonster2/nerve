# Nerve UI 🧠 - الارتجال الحركي السحابي بدون لمس

تطبيق ويب تجريبي من نوع تطبيقات الصفحة الواحدة (SPA) فائق الخفة والسرعة، يتيح للمستخدمين بناء جدار هندسي متكامل من المكعبات الرقمية بالاعتماد الكلي على إيماءات اليد وحركات الأصابع الحرة عبر الكاميرا بدون أي حاجة للمس الشاشة.

## 🛠️ الهندسة الفنية والتكنولوجية للمشروع

يعتمد التطبيق بالكامل على بنية تحتية لاحامية متطورة تضمن صفر تكاليف تشغيل وسرعة استجابة مذهلة:
1. **الرؤية الحاسوبية السحابية**: دمج فوري لمكتبات `MediaPipe Hands` لمعالجة بيانات الذكاء الاصطناعي مباشرة داخل المتصفح (Client-Side).
2. **الاستضافة الفولاذية**: يرفع المشروع على نظام **GitHub Pages** داخل بيئة مستقرة ومنفصلة.
3. **موجّه العمال (Cloudflare Workers)**: يتم ربطه وتوجيهه ديناميكياً تحت نطاق فرعي مخصص يحمل نفس اسم الهوية البرمجية لضمان تفعيل دروع التخزين المؤقت العالمي.

## 🕹️ ميكانيكا التفاعل العصبي

* **التحريك والمؤشر**: يتم التقاط رأس أصبع السبابة وتوجيهه ليكون هو الفأرة والمؤشر التفاعلي على الشاشة بعد محاكاته وعكسه هندسياً كمرآة لتسهيل التحكم.
* **القبض والترك (The Pinch)**: عند ضم أصبعي الإبهام والسبابة معاً، يتعرف النظام على حدث "القبض" ليتم الإمساك بالمكعب المتواجد في منطقة الدعم السفلى. فتح الأصابع يؤدي لترك المكعب فوراً.
* **الاصطفاف الشبكي (Grid Snapping)**: بمجرد إفلات المكعب، يقوم الكود بربطه ومحاذاته بأقرب خلية شبكية رياضية لضمان اصطفاف حائط البناء بشكل متناسق واحترافي.
